1. What is DOM and what is the purpose of DOM?

DOM stands for Document Object Model. This is a programming interface that represents the HTML document of a web page. JavaScript now creates all the elements on a web page and the relationships between elements. Generally, with JavaScript, element properties can be changed, new elements can be created.

2. How will you select HTML elements using DOM? Name the DOM methods.

The DOM is used to select something from within the HTML of a web page. For example, we can select an ID from HTML using document.getElementById, and we can select classes from HTML using document.getElementsByClassName. etc. we can use DOM in different ways to select different things and we can do different things with different changes in it.

3. What is event bubble?

Event bubbling is a mechanism in the Document Object Model (DOM). When an event bubble is applied to an element, it selects all elements nested within it, and so on. Clicking on any single element also clicks the rest of the nested elements.

4. What is a callback function and why will you use it?

A callback function is basically passed as a parameter to another function. Sometimes these functions are written to solve various problems, when needed, the function is done when it is needed, it does not need to write additional code and there is less patchiness in the code.
